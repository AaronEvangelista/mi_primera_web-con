##  Estrategia de diseño utilizada: Mobile First

Se implementó la estrategia **Mobile First**, donde los estilos base estan pensados para pantallas de los moviles  
Posteriormente, mediante **Media Queries**, el diseño se adapta a tabletas y escritorios.  

Ventajas de esta estrategia:
- Creo tener mejor rendimiento en moviles.  
- Codigo CSS más limpio y mantenible.  
- Experiencia de usuario consistente en todos los dispositivos.
